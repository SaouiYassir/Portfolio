## ✨ About The Project

This project is a **personal interactive CV and portfolio** designed to present my skills, projects, and experience in a visually engaging format. Built with modern web technologies, it combines professional content delivery with smooth animations and a responsive design.

### Key Features

- 🎨 **Modern, responsive UI** — Seamless experience across all devices
- ⌨️ **Dynamic typing animation** — Catchy typewriter effect on the hero section
- 🧭 **Smart navigation** — Intuitive navbar with smooth scrolling
- ⚡ **High performance** — Built with Vite for ultra-fast loading
- 🎬 **Engaging animations** — Smooth transitions and hover effects

## 🚀 Getting Started

To run this project locally, follow these steps.

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/SaouiYassir/SaouiYassir.github.io.git
   cd SaouiYassir.github.io
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

### Preview Production Build Locally

```bash
npm run preview
```

## 🎨 Customization

Want to use this template for your own portfolio? Here's how:

1. **Update personal information** — Modify content in the `Sections` components.
2. **Replace assets** — Add your own images to the `src/assets` directory.
3. **Adjust styling** — Tweak colors, fonts, and spacing in `src/index.css`.
4. **Add or remove sections** — Edit the main layout in `src/main.jsx`.
